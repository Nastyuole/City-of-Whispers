# City of Whispers

A text-based interactive fiction game that runs directly in your browser.  
Step into a mysterious world where your choices shape the story.

Demo:
https://nastyuole.github.io/City-of-Whispers/

---

## About the Game

City of Whispers is a narrative-driven experience where every decision matters.  
Explore a branching storyline, uncover secrets, and guide the outcome through your choices.

- Act 1 complete — 8,000+ words
- Multiple story paths and endings
- Choice-driven gameplay

---

## Features

- Branching narrative with meaningful choices
- 3 save slots to track progress
- Language support:
  - English
  - Bulgarian
  - Russian
- Adjustable font size
- Settings persist between sessions (no reload on language switch)

---

## Built With

- HTML, CSS, JavaScript
- React
- i18next (localization)
- JSON (translations)

---

## How to Run Locally

1. Clone the repository
2. Open the project folder in VS Code
3. Run index.html with Live Server
